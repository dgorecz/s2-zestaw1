# Prostszy zestaw

## Zadanie 1 – List<T> (operacje na liście)
Napisz program, który:
1. Tworzy List<int>.
2. Dodaje do listy liczby: 3, 7, 2, 9, 1, 5, 8, 6.
3. Wyświetla wszystkie elementy listy.
4. Sortuje listę rosnąco.
5. Usuwa wszystkie liczby mniejsze niż 5.
6. Dodaje na koniec liczby 10, 11, 12.
7. Oblicza i wyświetla:
o sumę elementów
o średnią elementów.

## Zadanie 2 – Dictionary<TKey, TValue> (zarządzanie danymi)
Utwórz program zarządzający ocenami studentów.
1. Utwórz Dictionary<string, List<int>>.
2. Dodaj studentów:
"Jan" → 3, 4, 5
"Anna" → 5, 5, 4
"Piotr" → 2, 3, 3
3. Wyświetl wszystkich studentów i ich oceny.
4. Oblicz średnią ocen dla każdego studenta.
5. Znajdź studenta z najwyższą średnią.

## Zadanie 3 – Queue<T> (symulacja systemu obsługi)
Stwórz program symulujący kolejkę w banku.
1. Utwórz Queue<string>.
2. Dodaj klientów:

Adam
Kasia
Marek
Ola
Tomek
3. Wyświetl osobę pierwszą w kolejce.
4. Obsłuż 2 osoby (Dequeue).
5. Dodaj nowych klientów: Paweł, Karolina.
6. Wyświetl aktualną kolejkę.
7. Wyświetl liczbę osób w kolejce.

## Zadanie 4 – Stack<T> (historia operacji)
Napisz program symulujący historię operacji w edytorze tekstu.
1. Utwórz Stack<string>.
2. Dodaj operacje:
"napisano tekst"
"dodano zdanie"
"usunięto słowo"
"dodano akapit"
3. Wyświetl ostatnią operację.
4. Cofnij dwie operacje (Pop).
5. Wyświetl wszystkie pozostałe operacje na stosie.
6. Wyświetl liczbę operacji w historii.

## Zadanie 5 – HashSet<T> (operacje na zbiorach)
Napisz program, który:
1. Tworzy dwa zbiory:
HashSet<int> A = {1,2,3,4,5,6}
HashSet<int> B = {4,5,6,7,8,9}
2. Wyświetla:
• sumę zbiorów (Union)

• część wspólną (Intersect)
• różnicę zbiorów (Except)
3. Sprawdza czy:
• zbiór A zawiera element 3
• zbiór B zawiera element 10

## Zadanie 6 – LinkedList<T> (operacje na liście dwukierunkowej)
Napisz program, który:
1. Tworzy LinkedList<string> z dniami tygodnia:
Poniedziałek
Wtorek
Środa
Czwartek
Piątek
2. Dodaje:
• Niedziela na początek
• Sobota na koniec
3. Wstaw "Święto" po Środzie.
4. Usuń element Wtorek.
5. Wyświetl wszystkie elementy listy.
6. Wyświetl pierwszy i ostatni element listy.
